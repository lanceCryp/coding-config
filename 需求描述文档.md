coding-config MVP 开发提示文档

目标：开发一个 Node.js CLI 工具，让开发者快速配置国产 AI 平台（智谱 GLM、阿里云百炼、火山引擎方舟）到主流 AI 编程工具（Cline、Cursor、Claude Code）。

项目信息

项目名称：coding-config
版本：0.1.0 (MVP)
技术栈：Node.js + TypeScript
CLI 框架：commander.js
交互框架：inquirer.js
开源协议：MIT

6.2 config/storage.ts

本地配置存储，用于记录用户历史配置：

typescript
import *as path from 'path';
import* as os from 'os';
import * as fs from 'fs-extra';

const CONFIG_DIR = path.join(os.homedir(), '.coding-config');
const CONFIG_FILE = path.join(CONFIG_DIR, 'config.json');

interface Config {
  platforms: Record<string, { apiKey: string }>;
  tools: Record<string, { platform: string }>;
}

export async function loadConfig(): Promise<Config> {
  try {
    return await fs.readJson(CONFIG_FILE);
  } catch {
    return { platforms: {}, tools: {} };
  }
}

export async function saveConfig(config: Config): Promise<void> {
  await fs.ensureDir(CONFIG_DIR);
  await fs.writeJson(CONFIG_FILE, config, { spaces: 2 });
}

export async function addPlatform(platformId: string, apiKey: string): Promise<void> {
  const config = await loadConfig();
  config.platforms[platformId] = { apiKey };
  await saveConfig(config);
}

export async function addTool(toolId: string, platformId: string): Promise<void> {
  const config = await loadConfig();
  config.tools[toolId] = { platform: platformId };
  await saveConfig(config);
}

6.3 config/validator.ts

API Key 格式验证：

typescript
export function validateApiKey(apiKey: string, platform: string): boolean {
  if (!apiKey || apiKey.length < 10) {
    return false;
  }

  // 智谱 GLM API Key 格式: xxx.yyy
  if (platform === 'zhipu') {
    return apiKey.includes('.');
  }

  // 阿里云百炼 API Key 格式: sk-xxxxx
  if (platform === 'ali') {
    return apiKey.startsWith('sk-');
  }

  // 火山引擎方舟 API Key 格式: 不特定格式，长度检查即可
  if (platform === 'volcengine') {
    return apiKey.length >= 20;
  }

  return true;
}

Windows: C:\Users\{username}\.vscode\settings.json
macOS/Linux: /Users/{username}/.vscode/settings.json
使用 os.homedir() 获取跨平台路径
